## github提交到仓库命令
1. cd..
2. git add 加关注
3. git commit 提交
4. git remote add origin 移到仓库
5. git push origin master 提交到仓库

## .c和.html修改完执行的命令

#### .c
1. make clean 清楚
2. make 编译
3. make install 提交

#### .html
sudo cp index.html /var/www/html 重新编译到/var/www/html下
